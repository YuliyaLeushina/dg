НАША СПЕЦИАЛИЗАЦИЯ

Back End — разработка

Разрабатываем нагруженные веб-приложения с использованием современных фреймворков и CMS. Проводим интеграции сайтов и интернет-магазинов с CRM, ERP, WMS, платёжными шлюзами, доставкой и другими внешними информационными системами. Разрабатываем API для взаимодействия с мобильными приложениями и frontend приложениями.

Front End — разработка

Верстаем статические макеты, реализуем анимацию, оптимизируем скорость загрузки сайтов. Разрабатываем SPA с использованием современных фреймфорков. Реализуем адаптивность под различные устройства, операционные системы и браузеры.

QA — тестирование

Повышаем качество разработки. Проводим функциональное, регресс, smoke тестирование. Автоматизируем процесс тестирования. Осуществляем автоматизирование PixelPerfect-тестирование верстки. Проводим нагрузочное тестирование.

Дизайн

Разработка концепции визуального стиля проекта, подготовка референсов, определение функционала будущего сайта. Отрисовка макетов для десктопной и мобильной версии, подготовка прототипов с возможность реального тестирования.

Поддержка

Осуществляем техническую поддержку проектов, обеспечиваем их непрерывное техническое развитие и доработки. Реанимируем проекты.
